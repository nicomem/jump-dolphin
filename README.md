# jump-dolphin

JUMP Dolphin project for EPIFIN

## Authors

- Nicolas Mémeint
- Thomas Curti
- Tom Méchineau

## Tested environment

The environment where the project have been tested on, other version might also work:

- CMake 3.18
- g++ 10.2

## How to build & run

```bash
# Build the project
mkdir build; cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j

# Run the binary
./dolphin
```
